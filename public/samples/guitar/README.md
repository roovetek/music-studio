# Local guitar one-shots (optional)

Add **short** mono or stereo **WAV** files here if you later wire a sample engine (not enabled by default).

Suggested filenames (future loader):

- `strum-down.wav` — downstroke chord
- `strum-up.wav` — upstroke
- `ghost-mute.wav` — palm mute / choked strum
- (No file for **rest** — engine uses silence.)

**Do not** commit large copyrighted samples. Use **CC0** or cleared recordings only. See [docs/guitar-samples.md](../../docs/guitar-samples.md) for where to find suitable sounds.

The `.gitkeep` in this folder keeps the directory in git so adding files is straightforward.
